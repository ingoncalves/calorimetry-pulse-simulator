Detailed Description

A struct that contains a sliced dataset.

This struct contains a sliced dataset. The dataset is composed of three matrices: the samples matrix, the ampltitudes matrix, and the phases matrix. Each matrix has a number of rows equal to the number of pulses in the dataset, and a number of columns equal to the slice size. The samples matix contains the sample coordinate of the signal at a given window index. The amplitude vector contains the truth amplitude value at a given window index, and the phase vector contains the truth phase value at a given window index.
