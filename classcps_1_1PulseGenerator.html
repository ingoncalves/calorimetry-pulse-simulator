Detailed Description

A pulse generator that generates analog pulses.

This class generates analog pulses. The pulse generator has a pulse shape, which is the mathematical function that describes the shape of the pulse. The pulse generator also uses random distributions to generate values for the following pulse properties:

  • amplitude;
  • phase;
  • deformation;
  • noise.
